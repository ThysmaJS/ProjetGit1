Ici seront détaillé mes commandes : 

1/ git init --> initier le repertoir GIT
2/ git add -A --> Ajout des fichiers de base (f1, f2 et commande)
3/ git commit -m ( mon message ) --> Premier commit qui initie le projet 
4/ git commit -m ( mon message ) --> Modification des fichiers F1 et F2 
5/ git add -A --> Ajout des fichiers demandé (f3 et f4)
6/ git commit -m ( mon message ) --> Ajout des fichiers demandé f3 et f4
7/ git add -A --> Ajout des fichiers demandé (gitignore)
8/ git commit -m ( mon message ) --> Ajout des fichiers demandé + ignore f3 et f4
7/ git add -A --> Ajout des fichiers demandé (f5 et f6)
8/ git commit -m ( mon message ) --> Ajout des fichiers demandé f5 et f6
9/ git branch A --> Créer la branche "A"
10/ git branch B --> Créer la branche "B"
11/ git checkout A --> Allez sur la branche A 
12/ git commit -m ( mon message ) --> Modification du f1 sur la branche A
